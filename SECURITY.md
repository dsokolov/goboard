# Security Policy

## Supported versions

Security fixes are provided for the latest release published on [GitHub Releases](https://github.com/dsokolov/goboard/releases).

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues through [Obsidian's security reporting process](https://help.obsidian.md/resources#Report+a+security+issue) or contact the maintainer via GitHub with minimal details so we can coordinate a fix before disclosure.

Include:

- A description of the issue and its impact
- Steps to reproduce
- Obsidian and GoBoard versions
- Any proof-of-concept if available

We aim to acknowledge reports within a reasonable time and will publish a fix in a new release when confirmed.
