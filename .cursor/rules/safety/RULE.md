После внесения изменений не забывай проверять что:
1) проект собирается
2) тесты проходят
3) проходит линтер

Команды для сборки пароекта, для запуска тестов и для запуска линтера смотри в README.md

Если падают тесты, то определись, это тесты устарели или сломался код. Если устарели тесты - обнови их. Если сломался код - почини.
