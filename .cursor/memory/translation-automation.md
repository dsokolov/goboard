# Автоматизация переводов документации

## Что было настроено

**Правило автоматической синхронизации** (`.cursor/rules/translation_sync.md`)
- Указывает AI автоматически проверять и обновлять переводы при работе с проектом
- Описывает процесс обновления английских файлов на основе русских
- AI автоматически обновляет переводы без дополнительных запросов

## Как это работает

AI автоматически проверяет переводы при каждом взаимодействии с проектом:
- Находит все файлы с суффиксом `-RU.md` или `-ru.md`
- Для каждого русского файла находит соответствующий английский файл
- Сравнивает содержимое и даты модификации
- Если русская версия обновлена, автоматически обновляет английскую версию

## Пары файлов
- `README-RU.md` → `README.md`
- `docs/examples-ru.md` → `docs/examples.md`
