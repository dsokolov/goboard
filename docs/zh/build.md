# 构建

语言:&nbsp;[English](../en/build.md)&nbsp;•&nbsp;[Русский](../ru/build.md)&nbsp;•&nbsp;**中文**

本文档介绍如何从源代码构建 GoBoard 插件。

构建项目，包括将分发文件复制到开发仓库：

```
npm run build
```

运行测试：

```
npm test
```

运行代码检查：

```
npm run lint
```

更新测试基线：

```
npm run update-baseline
```

更新文档中的图片：

```
npm run update-docs
```
