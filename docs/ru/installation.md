# Установка

Язык:&nbsp;[English](../en/installation.md)&nbsp;•&nbsp;**Русский**&nbsp;•&nbsp;[中文](../zh/installation.md)

Как установить плагин GoBoard? Для этого существует несколько способов.

## Через приложение Obsidian (рекомендуется)

1. В приложении Obsidian откройте настройки плагинов сообщества: `Settings -> Community plugins`.
2. Если вы этого ещё не делали, нужно будет включить поддержку сторонних плагинов `Turn on community plugins`.
3. Перейти к выбору плагинов, нажав `Browse`.
4. В строке поиска ввести `GoBoard`.
5. Выбрать плагин, нажать `Install`, а после завершения установки - `Enable`.
6. Готово, теперь можно рисовать диаграммы!

## С помощью BRAT

Если по каким-то причинам вам не подходит установка через интерфейс Obsidian,
вы можете установить GoBoard через [BRAT](https://github.com/TfTHacker/obsidian42-brat).

1. Убедитесь, что у вас установлен [BRAT](https://github.com/TfTHacker/obsidian42-brat). Если нет - [установите его](https://obsidian.md/plugins?search=BRAT).
2. В настройках плагина BRAT добавьте репозиторий `https://github.com/dsokolov/goboard`, следуя [инструкции](https://github.com/TfTHacker/obsidian42-brat).
3. Готово, теперь можно рисовать диаграммы!

## Вручную

1. Выберите [подходящий релиз](https://github.com/dsokolov/goboard/releases) и скачайте из него 3 файла: `main.js`, `manifest.json` и `styles.css`.
2. Скопируйте эти файлы в ваше хранилище, в каталог `<корень_хранилища>/.obsidian/plugins/goboard/`. Если таких каталогов нет - создайте их.
3. В настройках Obsidian включите поддержку плагинов сообщества (если ещё не была включена).
4. В списке плагинов найдите и включите `GoBoard`. Возможно, потребуется перезапуск Obsidian.
5. Готово, теперь можно рисовать диаграммы!
