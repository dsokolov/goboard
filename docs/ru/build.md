# Сборка

Язык:&nbsp;[English](../en/build.md)&nbsp;•&nbsp;**Русский**&nbsp;•&nbsp;[中文](../zh/build.md)

В этом документе описана сборка плагина GoBoard из исходников.

Собрать проект, включая копирование дистрибутива в хранилище для разработки:

```
npm run build
```

Запуск тестов:

```
npm test
```

Запуск линтера:

```
npm run lint
```

Обновление бейзлайна тестов:

```
npm run update-baseline
```

Обновление картинок в документации:

```
npm run update-docs
```
