# Мини-игра 5x5

Пример очень маленькой доски для демонстрации или обучения детей.

```goboard
size 5x5
coordinates on
W D4
W B4
W D3
W D2
W B2
W B3
W C3
W A1
W B1
W C2
W D1
B C5
```

## Описание

- Доска 5x5 - минимальный размер для игры
- Идеально для демонстрации правил
- Очень быстрые игры
- Подходит для детей и начинающих
