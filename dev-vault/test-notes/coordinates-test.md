# Тест координат

Проверка работы координат на разных размерах досок.

## Доска 9x9 с координатами

```goboard
size 9x9
coordinates on
B D4
B F4
B G8
W E8
```

## Доска 19x19 с координатами (по умолчанию)

```goboard
size 9x9
coordinates on
B D4
B F4
B G8
W E8
```

## Доска 9x9 без координат

```goboard
size 9x9
coordinates on
B D4
B F4
B G8
W E8
```
