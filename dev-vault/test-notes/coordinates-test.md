# Тест координат

Проверка работы координат на разных размерах досок.

## Доска 9x9 с координатами

```goboard
size 9x9
coordinates on
B D4
W D6
B F4
W F6
```

## Доска 19x19 с координатами (по умолчанию)

```goboard
size 19x19
B D4
W D16
B Q4
W Q16
```

## Доска 9x9 без координат

```goboard
size 9x9
coordinates off
B D4
W D6
B F4
W F6
```

## Доска 5x5 с координатами

```goboard
size 5x5
coordinates on
B C3
W C4
B D3
W D4
```
